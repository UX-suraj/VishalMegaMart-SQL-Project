# Retail Store Sales Analysis using SQL

## Objective
Analyze a retail store dataset using SQL to answer real-world business questions.

## Tools Used
- SQL (MySQL / PostgreSQL)

## Database Tables
- customers
- products
- orders
- order_items

## Key Analysis Performed
- Total sales per order
- Customer-wise total spending
- Best-selling products
- Category-wise sales analysis

## Skills Demonstrated
- SQL Joins
- Group By & Aggregations
- Filtering & Sorting
